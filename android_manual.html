1

앱 설치

APK 다운로드 — 공식 사이트에서 최신 APK 파일을 다운로드합니다.
설치 허용 — 설치가 차단되면 설정 → 보안 → 알 수 없는 앱 설치 허용을 활성화하세요.
그래도 안 된다면 — 보안 앱의 자동 위협 차단 기능을 일시적으로 꺼주세요.
2

앱 실행

앱 실행 — 설치가 완료되면 전산볼트 앱을 실행합니다.
권한 허용 — 요청되는 권한은 허용해 주세요.
알림 권한 — 알림은 필수 권한이 아닙니다. 거부해도 앱 사용에 문제 없습니다.

4

자주 묻는 질문

Q 설치가 안 돼요.
설정 → 보안에서 알 수 없는 앱 설치를 허용해 주세요. 그래도 설치되지 않으면 보안 앱의 자동 위협 차단 기능을 일시적으로 꺼주세요.
Q 앱이 실행되지 않아요.
기기의 Android 버전이 최소 요구 사양을 충족하는지 확인해 주세요. 문제가 지속되면 디스코드 서버를 통해 문의해 주세요.
Q 바이러스 경고가 뜨는데 어떻게 하나요?
AXLIB 공식 앱에는 바이러스가 없습니다.

보안 앱이 오탐지하는 경우라면 무시하고 설치를 진행하셔도 됩니다.
만약 공식 사이트 외 출처에서 다운받은 앱에서 경고가 뜬다면 즉시 삭제 후 공식 사이트에서 다시 다운로드하세요.
공식 사이트에서 받은 파일에서도 바이러스가 감지된다면 빠르게 수정하겠습니다.
