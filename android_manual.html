1. 앱 설치

다운로드부터 시작하는 가장 기본적인 단계입니다.

1
APK 다운로드 - 공식 사이트에서 최신 APK 파일을 다운로드합니다.
다운로드 열기
2
보안 검사 - 설치 전에 보안 앱으로 파일을 한 번 검사해 보시는 것을 권장합니다.
3
설치 허용 - 설정 → 보안 → 알 수 없는 앱 설치 허용을 활성화하세요.
4
그래도 안 된다면 - 공식 사이트에서 받은 파일인지 다시 확인하고, 계속 문제면 디스코드로 문의해 주세요.

2. 앱 실행

설치가 끝나면 권한과 초기 설정을 확인합니다.

1
앱 실행 - 설치가 완료되면 전산볼트 앱을 실행합니다.
2
권한 허용 - 요청되는 권한은 허용해 주세요.
3
알림 권한 - 알림은 필수 권한이 아닙니다. 거부해도 앱 사용에 문제가 없습니다.

4. 자주 묻는 질문

자주 막히는 부분만 먼저 정리했습니다.

Q 설치가 안 돼요.
설정 → 보안에서 알 수 없는 앱 설치를 허용해 주세요. 그래도 설치되지 않으면 공식 사이트에서 받은 파일이 맞는지 확인하신 뒤, 디스코드 문의 채널로 알려주세요.
Q 앱이 실행되지 않아요.
기기의 Android 버전이 최소 요구 사양을 충족하는지 확인해 주세요. 문제가 지속되면 디스코드 서버를 통해 문의해 주세요.
Q 바이러스 경고가 뜨는데 어떻게 하나요?
Play 스토어 외부에서 설치하는 앱은 보안 프로그램에서 주의 알림이 표시될 수 있습니다. 항상 공식 사이트에서 받은 파일인지 먼저 확인해 주세요.

설치 전에 보안 앱으로 한 번 검사해 보시는 것을 권장합니다. 출처가 확실하지 않은 곳에서 받은 파일이라면 설치하지 마시고 바로 삭제해 주세요. 공식 사이트에서 받은 파일에서도 경고가 계속된다면, 보안 기능을 끄지 마시고 디스코드로 알려주세요.
