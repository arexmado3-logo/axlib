## AXLIB 소개

**AXLIB**는 소프트웨어, 앱, 웹 서비스를 만드는 개발 플랫폼입니다.

현재 주요 프로젝트는 **전산볼트 앱**이며,  
현장에서 사용할 수 있는 볼트 계산, 재고 관리, 작업 기록 기능을 목표로 합니다.
