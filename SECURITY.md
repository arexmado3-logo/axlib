# Security Policy

## Supported Versions

현재 지원 중인 버전입니다.

| Version | Supported |
| ------- | --------- |
| Latest  | ✅ |
| Older Versions | ❌ |

## Reporting a Vulnerability

AXLIB 프로젝트의 보안 취약점을 발견한 경우 공개적으로 게시하지 말고 아래 방법으로 제보해 주세요.

### Contact

- Discord: https://discord.gg/DxY47Q3EHR

### What to Include

가능하다면 아래 정보를 포함해 주세요.

- 취약점 설명
- 재현 방법
- 영향 범위
- 스크린샷 또는 로그
- 사용 중인 버전

### Response Process

1. 취약점 제보 접수
2. 내용 검토
3. 수정 여부 결정
4. 수정 후 업데이트 배포

### Responsible Disclosure

사용자 보호를 위해 수정 전까지 취약점을 공개하지 말아 주세요.

감사합니다.

AXLIB Security Team
